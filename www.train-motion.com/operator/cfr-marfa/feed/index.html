]]> This is a REPAINT/RESKIN. That means that the package contains only textures and other small improvements added to the main loco. In order for this asset to work you need to have installed the main model of CFR 060 – EA 476-692-5 which can be found
HERE.

 

Note: This asset is depending on a licensing system that requires an original copy of Train Simulator game and a valid Steam account.

!  Train Simulator must be installed in default Steam directory: !
“Steam\steamapps\common\RailWorks\”

 

Installation
Step 1 – Make sure that you have the main model (CFR 060 – EA 476-692-5) of the loco installed and working.
Step 2 – Make sure Train Simulator is installed in the same location as the Steam program. (NOT in SteamLibrary or another folder)
Step 3 – Download and install this package.

 

History

The 40 class locomotive is one of the most recent modernised versions of 060 EA originals, built for freight trains. It is among the fewest locomotives that have blue taillights and ICOL TSAM modernization. It can reach only up to 100kph but has a very strong torque which makes it very well fitted for long freight trains.

 

Technical specifications
Type: Electric
Axle formula: CoCo
Weight: 120 t (264,554.7 pounds)
Length: 19,800 mm (64.96 ft)
Engine Power: 5,100 kW (6,800 hp)
Max Speed: 100 km/h (62 mph)

 

Key Features

New hornsounds
HD real exterior textures
HD winter exterior textures
HD interior textures
New headlights
New blue color taillights
Modified interior camera
